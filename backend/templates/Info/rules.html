Правила

  1. Состав команд.

    В каждой команде должно быть 4 пятиклассника, 4 шестиклассника, 4 семиклассника, 4 восьмиклассника и 4 девятиклассника. Придерживаемся формирования команд по вертикалям (например «5а», «6а», «7а», «8а», «9а»). В случае, если в классе нет достойных желающих для участия в команде, то можно докомплектовать команду учениками любого класса параллели. Команды аертикалей «Е» и «И» могут дополняться учащимися разных классов.

  2. Открытие ИТИ.

    На открытии у команд должен быть отличительный знак, название и кричалка.

    • ИТИ-2020. В качестве отличительного знака были предложены маски на лицо, рассписанные особенным образом. Маски обязательно должны быть рекомендованы детям на всё время проведения ИТИ.

  3. Запасные участники.

    Каждая команда в кадой параллели набирается по одному дополнительному игроку. Псоле руководитель составляет подневный список «отдыхающих» игрноков (баллы, полученные ими, не идут в зачёт команды).

    • Действовало только в ИТИ-2019.

  4. Замены в командах.

    Замену в команде можно производить только по причине болезни участника команды, подтвержденной документально. Замена выбирается из числа школьников, не участвующих в других командах (нельзя забрать ребёнка из «6м» в вертикаль «М», если он уже зарегистрирован в команде «И»).

  5. Индивидуальный тур.

    Участвуя в индивидуальных турах, каждый участник работает на индивидуальный рейтинг, на рейтинг своего класса и своей команды. !Оценивание тура!. В зачёт идёт два лучших результата за день !четыре за 2 дня?!.

  6. Груповой тур.

    В групповом туре участвуют группы от 2 до 6 человек (все из одной команды). Команды обязаны приходить в полном составе (при неполном составе команда может участвовать в туре, но получает за него 0 баллов).

    Начисление баллов:

    • ИТИ-2019. Победитель каждого тура получает 800 баллов, шаг между местами — 100 баллов.

    • ИТИ-2020. Пусть n — число участников тура. Тогда победитель получает 200n баллов, а шаг между местами составляет 25n баллов.

    Список туров:

    • Карусель — полипредметная олимпиада (математика, словесность, естествознание).

    • Естественно-научные бои — турнир по физике, математике, биологии, химии.

    • Коммуникативные бои

    • Парный тур по литературе

    • Парный тур по английскому языку

    • Театр

    • Изобретательство

  7. Командный тур.

    Присутствует вся команда, в случае не полного состава команда получает 0 баллов за командный тур.

    Начисление баллов:

    • ИТИ-2019. Победитель получает 5600 баллов, шаг между местами — 700 баллов.

    • ИТИ-2020. Победитель получает 4000 баллов, шаг между местами — 500 баллов.
